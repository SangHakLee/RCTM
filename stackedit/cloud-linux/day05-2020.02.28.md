## chroot
> 특정 디렉토리를 최상위 `/`처럼 보이게 변경

```bash
# host
$ mkdir -p ~/newroot/bin
$ mkdir -p ~/newroot/lib64

$ chroot ~/newroot /bin/bash # 
chroot: failed to run command ‘/bin/bash’: No such file or directory



```
